Django中使用mysql数据库

三 18 六月 2014

安装Mysql

sudo apt-get install mysql-server
sudo apt-get install python-mysqldb
sudo apt-get install libmysqlclient-dev

安装mysql-python模块

pip install mysql-python

创建并设置Database

mysql -u root -p
Enter password: 

mysql> CREATE DATABASE tulpar_db;
