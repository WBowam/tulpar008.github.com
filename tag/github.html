git 取消错误的上传

Tue 23 July 2013

参考Stackoverflow

通过如下命令显示最近的上传情况

git rebase -i HEAD~3

将会显示最近的2个上传，也可以显示任意个，如HEAD~10
pick d877e57 %s
pick f032eda %s
