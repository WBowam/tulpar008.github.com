部署Socket.io 的demo

Wed 11 June 2014

1.安装nvm

$ wget -qO- https://raw.github.com/creationix/nvm/v0.4.0/install.sh | sh

2.安装node

nvm install 0.1
若过程中如果报错，说：创建目录/usr/sbin/src 权限不够

我用了个极端的方法来解决此问题，不推荐

sudo chmod 777 /usr/sbin/

3. 跑起

cd socket.io/example/chat/
node index.js
